# Git Guide

## Setup Guide

1. Install git (if not already installed)  
 <b>In your IDE's terminal:</b>
2. `git clone https://github.com/OmarAhmedTHE25th/Compiler_Project`
3. `cd Algorithms-Project`
4. `git config --global user.name "Your Name"` (Github name)
5. `git config --global user.email "your@email.com" ` (Account's email)
